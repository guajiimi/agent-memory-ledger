# Submission Description

I built Agent Memory Ledger, Memory curation helper for AI assistants and long-running agents. The project relates directly to AI agents because it converts agent workflow data into structured JSON artifacts that a coding agent or human reviewer can use. It solves this problem: Long-running agents need durable memory, but temporary task state and stale facts should not pollute future sessions. Core logic includes JSON input parsing, agent-oriented analysis, deterministic reports, and a static demo dashboard for proof of usage.
